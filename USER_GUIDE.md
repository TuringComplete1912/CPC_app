# 学生第六党支部数字化工作平台 - 用户操作指南

## 📱 平台简介

学生第六党支部数字化工作平台是一个集文档管理、活动记录、学习资料共享、社区交流和党务管理于一体的综合性党建工作平台。

---

## 🚀 快速开始

### 注册与登录

#### 注册账号
1. 访问注册页面
2. 填写用户名和密码（至少6位）
3. 输入邀请码：
   - `branch6` - 普通用户
   - `admin6` - 管理员（自动获得管理员权限）
4. 点击"注册"完成

#### 登录
1. 输入用户名和密码
2. 点击"登录"进入工作台

---

## 📋 功能板块详解

### 1. 工作台（首页）

**功能概述**：
- 查看近期文档和活动日志的快速入口
- 全局搜索功能
- 快速访问各个功能模块

**如何使用**：
1. 登录后自动进入工作台
2. 点击搜索框可搜索社区、文档、学习资料
3. 查看近期文档和活动日志的摘要
4. 点击"查看全部"进入对应模块

---

### 2. 社区

**功能概述**：
- 发起话题讨论
- 回答问题
- 点赞和回复
- 查看其他成员的党员身份和班级

**如何使用**：

#### 创建话题
1. 点击顶部导航栏"社区"
2. 点击"添加话题"按钮
3. 填写话题标题和描述
4. 点击"确认"发布

#### 参与讨论
1. 点击话题卡片进入详情页
2. 在"写回答"区域输入你的回答
3. 点击"提交回答"发布
4. 可以点赞其他人的回答
5. 可以回复其他人的回答

#### 查看回复
1. 点击回答下方的回复数量按钮
2. 展开查看所有回复
3. 点击"回复"按钮添加新回复

**权限说明**：
- 所有用户可以创建话题和回答
- 只能删除自己的话题和回答
- 管理员可以删除任何话题和回答
- 回答按点赞数降序排列（高赞优先）

---

### 3. 近期文档

**功能概述**：
- 创建和编辑支部文档
- 多人协作编辑
- 文档发布和草稿管理
- 上传PDF附件
- 导出PDF并AI优化

**如何使用**：

#### 创建文档
1. 点击"近期文档"
2. 点击"新建文档"
3. 输入文档标题和内容
4. 点击"保存"

#### 编辑文档
1. 点击文档卡片进入详情页
2. 点击"编辑"按钮（仅草稿状态可编辑）
3. 修改内容后点击"保存"

#### 发布文档
1. 在文档详情页点击"发布"
2. 确认后文档变为已发布状态
3. 已发布的文档无法再编辑

#### 上传附件
1. 在文档详情页（草稿状态）
2. 点击"上传附件"
3. 选择PDF文件（最大50MB）
4. 上传成功后可以下载或删除

#### 导出PDF
1. 点击文档卡片上的导出按钮
2. 选择"直接导出"或"AI优化后导出"
3. AI优化会自动润色文档内容
4. 下载生成的PDF文件

**权限说明**：
- 所有用户可以创建文档
- 只能编辑自己创建的草稿文档
- 管理员可以删除任何文档
- 已发布的文档无法编辑和上传附件

### 5. 学习园地

**功能概述**：
- 理论学习和课程学习两个分类
- 上传和下载学习资料
- 支持PDF、PPT、ZIP文件
- 分类管理

**如何使用**：

#### 查看学习资料
1. 点击"学习园地"
2. 切换"理论学习"或"课程学习"标签
3. 点击分类查看该分类下的资料

#### 上传资料
1. 进入某个分类
2. 点击"上传资料"
3. 选择文件（PDF、PPT或ZIP，最大50MB）
4. 上传成功后显示在列表中

#### 下载资料
1. 点击资料卡片上的下载按钮
2. 文件自动下载到本地

#### 创建分类
1. 在学习园地页面点击"添加分类"
2. 输入分类名称和描述
3. 选择类型（理论学习或课程学习）
4. 点击"确认"创建

**权限说明**：
- 所有用户可以上传和下载资料
- 所有用户可以创建课程学习分类
- 只有管理员可以删除理论学习分类
- 理论学习分类包含三个预设分类：
  - 马克思列宁主义
  - 毛泽东思想
  - 习近平新时代中国特色社会主义思想

### 6. 党务管理

**功能概述**：
- 填写和管理个人党务信息
- 查找同年/同阶段同志
- 管理员可查看所有用户信息

**如何使用**：

#### 普通用户

**填写党务信息**：
1. 点击"党务管理"
2. 填写个人信息：
   - 政治面貌（群众/共青团员/入党积极分子/预备党员/党员）
   - 班级（格式：土木2201）
   - 籍贯
   - 微信号/QQ号
   - 各阶段时间（根据政治面貌自动显示）
3. 设置每个字段的公开/隐私状态
4. 点击"保存信息"

**查找同志**：
1. 选择"同年同志"或"同阶段同志"
2. 点击"搜索"
3. 查看搜索结果（只显示公开信息）
4. 点击卡片查看详细信息

#### 管理员

**访问管理员页面**：
- 管理员访问"党务管理"会自动跳转到管理员页面

**筛选排序**：
1. 点击"按姓名首字母"：按姓氏拼音排序
2. 点击"按年份"：按班级年份排序
3. 点击"按发展阶段"：按党员发展阶段排序

**搜索用户**：
1. 在搜索框输入关键字
2. 可搜索：姓名、班级、政治面貌、籍贯
3. 按回车或点击搜索按钮

**查看详情**：
1. 点击用户卡片
2. 查看完整党务信息（包括隐私信息）
3. 点击"返回列表"回到搜索结果

**权限说明**：
- 普通用户只能查看自己的信息和其他用户公开的信息
- 管理员可以查看所有用户的所有信息（包括隐私）
- 搜索结果在会话中保留，刷新页面会重置

### 7. 个人主页

**功能概述**：
- 管理个人资料
- 设置API Key
- 与学六小助手对话

**如何使用**：

#### 编辑个人资料
1. 点击"个人主页"
2. 点击"编辑资料"按钮
3. 修改昵称、部门、简介、电话、邮箱
4. 点击"保存"

#### 设置API Key
1. 点击"设置 API Key"按钮
2. 输入你的 OpenRouter API Key
3. 勾选"优先使用我的API Key"（可选）
4. 点击"保存"

**说明**：
- 如果不设置个人API Key，将使用系统默认Key
- 设置后可以选择是否优先使用个人Key
- 可以随时清除个人Key

#### 与AI助手对话
1. 在聊天框输入问题
2. 按回车或点击发送按钮
3. AI助手会回答你的问题
4. 可以点击"查看思考过程"查看AI的推理过程

**AI助手功能**：
- 询问支部的文档、活动、资料等信息
- 获取党建知识和政策解答
- 支持流式输出，实时显示回答
- 显示AI的思考过程（DeepSeek R1模型）

---

### 8. 全局搜索

**功能概述**：
- 搜索社区话题、文档、学习资料和分类
- 按类型筛选结果
- 快速跳转到详情页

**如何使用**：

1. 点击工作台的搜索框
2. 输入关键字
3. 按回车或点击搜索按钮
4. 查看搜索结果
5. 点击"全部"、"社区"、"文档"、"资料"、"分类"筛选
6. 点击结果卡片跳转到详情页
7. 浏览器返回时保持搜索结果

**搜索范围**：
- 社区话题：标题和描述
- 文档：标题和内容摘要
- 学习资料：标题
- 分类：名称和描述

## 👥 用户身份显示

在社区、文档、活动日志中，用户名旁会显示：
- **党员身份徽章**：党员/预备党员/积极分子/团员/群众
- **班级徽章**：如"土木2201"

**徽章颜色**：
- 党员：红色
- 预备党员：橙色
- 入党积极分子：黄色
- 共青团员：蓝色
- 群众：灰色

**隐私控制**：
- 只显示用户设置为公开的信息
- 未公开的信息不会显示徽章

## 💡 使用技巧

### 1. 文档协作
- 多人可以同时编辑同一个草稿文档
- 系统会记录所有编辑者
- 发布前确保内容完整

### 2. 学习资料管理
- 使用清晰的文件名
- 选择正确的分类
- 定期整理和更新资料

### 3. 党务信息填写
- 班级格式必须标准化：专业+年份（如：土木2201）
- 根据政治面貌填写对应的时间节点
- 合理设置隐私选项

### 4. AI助手使用
- 提问要具体明确
- 可以询问支部相关的任何信息
- 查看思考过程了解AI的推理逻辑

### 5. 搜索技巧
- 使用关键字搜索
- 利用类型筛选快速定位
- 浏览器返回保持搜索状态

## 📞 常见问题

**Q: 忘记密码怎么办？**
A: 联系管理员重置密码

**Q: 如何成为管理员？**
A: 使用管理员邀请码 `admin6` 注册，或联系现有管理员

**Q: 文档发布后可以修改吗？**
A: 不可以，发布后的文档无法修改，请在发布前仔细检查

**Q: 上传的文件有大小限制吗？**
A: 有，PDF附件最大50MB，学习资料最大50MB

**Q: 党务信息会被其他人看到吗？**
A: 只有你设置为公开的信息才会被其他普通用户看到，管理员可以查看所有信息

**Q: AI助手的回答准确吗？**
A: AI助手基于支部的文档和资料回答，但仍可能出错，重要信息请以官方文档为准
