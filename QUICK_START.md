## ❓ 常见问题

**Q: 如何添加更多管理员？**
A: 编辑 `prisma/set-admins.ts`，在 `adminUsernames` 数组中添加用户名

**Q: 数据会丢失吗？**
A: 不会！所有数据保存在本地 SQLite 文件中

**Q: 如何重置数据库？**
A: 删除 `prisma/dev.db` 文件，然后运行 `npx prisma migrate dev`

**Q: 忘记管理员密码怎么办？**
A: 可以通过数据库直接修改，或重新注册后再次运行设置脚本
